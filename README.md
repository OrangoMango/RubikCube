# RubikCube
Rubik's cube made with javafx3D
# Features
* Custom moves
* Custom algorithms
* **Solving algorithm using the layers method** *(3x3 cube only)*
* any NxNxN cube
# How to run
* JDK 17+ needed
```bash
./gradlew run
```
